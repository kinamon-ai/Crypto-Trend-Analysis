# Bitcoin Trend Analysis Tool 📈

「アノマリーサイクル」ロジックに基づいたビットコイン（BTC）のトレンド分析ツールです。
Streamlitを使用したWebインターフェースで、マルチタイムフレーム分析、アノマリーサイクルの確認、トレードシグナルの検知が可能です。

## 機能

*   **アノマリー分析**: 4年サイクルに基づいた現在の市場フェーズ判定
*   **マルチタイムフレーム分析**: 月足、週足、日足、4時間足、1時間足の一括トレンド判定
*   **シグナル検知**: リターンムーブ、MACDクロスなどの高確率セットアップの検知
*   **インタラクティブチャート**: Plotlyを使用した詳細チャート（MA、MACD表示）

## インストール方法

1.  リポジトリをクローンします。
    ```bash
    git clone <repository-url>
    cd <repository-dirname>
    ```

2.  必要なライブラリをインストールします。
    ```bash
    pip install -r requirements.txt
    ```

## 実行方法

以下のコマンドでアプリケーションを起動します。

```bash
streamlit run app.py
```

ブラウザが自動的に開き、ダッシュボードが表示されます。

## ロジックについて

*   **トレンド**: 価格がSMA200およびSMA100の上にあれば上昇、下にあれば下降と判断します。
*   **リターンムーブ**: 価格がEMA20付近まで戻り、かつMACDが反転の兆しを見せている場合にシグナルを出します。
