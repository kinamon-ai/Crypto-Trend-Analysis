# 相場サイクル・完全統合型アルゴリズム仕様書

## 1. 環境認識レイヤー（フィルター条件）
トレードを開始する前に、まず「戦うべき方向」を固定します。

*   **アノマリー・フィルター**: 「3年上昇・1年下落」の長期サイクルを基盤とし、現在のフェーズに応じて売買バイアス（ロング優先かショート優先か）を決定する。
*   **マルチタイムフレーム（MTF）分析**: 上位足（月足・週足・日足）の方向性を最優先とし、下位足（4時間足以下）でエントリーのタイミングを測る。
*   **時間帯・ボラティリティ制限**: 米国市場開始直後の急変を確認してからエントリーし、ボラティリティが異常に高い場合はロットを通常の1/10〜1/100に落とす。

## 2. テクニカル・セットアップ
使用するインジケーターの数値を統一し、ラインを自動描写します。

*   **移動平均線 (MA)**: 
    *   SMA 7, 25, 100（長期線）, 200（主要基準線）。
    *   EMA 20（リターンムーブの目安）。
*   **MACD**: デフォルト設定 (12, 26, 9) を使用。
*   **自動水平ライン**: 直近の主要な高値・安値をベースにレジサポラインを自動で引く。

## 3. エントリーロジック（パワーアップ版）
単なるブレイクではなく、**「テクニカルの否定」と「レジサポ転換」**を組み合わせた高精度な条件です。

### A. リターンムーブ確定エントリー
1.  **ブレイクの確認**: 主要ライン（200MAや直近高安）をローソク足の実体で明確に抜ける。
2.  **リターン（戻り）の待機**: ライン突破後、再度そのラインまたは20MA付近まで価格が押し戻されるのを待つ。
3.  **MACDゾーン・フィルター**: 
    *   **ロング**: MACDが「マイナス圏の深い位置」でゴールデンクロス(GC)すること（ゼロライン付近の空振りGCは無視）。
    *   **ショート**: MACDが「プラス圏の高い位置」でデッドクロス(DC)すること。
4.  **反転サイン**: ラインタッチ時に上髭・下髭などの反転の兆しを確認してエントリー。

### B. 期待値を高める補助ロジック
*   **「3」の法則**: 同一ラインへの3回目のタッチを、ライン突破（または反転）の強力なサインとして検知する。
*   **トラップ（お箱）回避**: レンジブレイク直後に直近高安を一瞬更新してから逆行する「ブルトラップ・ベアトラップ」を確認後の反転を狙う。
*   **ダイバージェンス否定**: 価格とMACDの逆行（ダイバーケンス）が発生し、それが否定された際の強いトレンド回帰に乗る。

## 4. 決済およびリスク管理（鉄の掟）
「予想ではなく計画」に基づき、感情を排除した決済を行います。

*   **損切り (Stop Loss)**: 
    *   固定％での損切りは行わず、**「エントリーの根拠（MAやライン）を実体で逆方向に抜けた時」**に即執行する。
*   **利確 (Take Profit)**: 
    *   「頭と尻尾はくれてやれ」の精神で、次の主要レジサポラインの手前で分割利確する。
    *   MACDの逆行クロス確認での反転決済。
*   **負けゼロ（Break Even）戦略**: 
    *   含み益が一定（例：1%以上）に達した時点で、ストップロスを**建値（プラマイゼロ）**に移動させる。
*   **資金管理**: 
    *   1回の最大損失を口座残高の一定割合（例：2%）以内に抑える。
    *   利益は定期的に別口座へ出金し、万が一のロスカットに備える「防御型」の運用を行う。
